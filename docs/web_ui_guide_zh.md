# Web UI 启动与使用说明

本说明覆盖 Web 前端的启动与基础使用，适用于项目根目录下的内置 Web UI（静态资源位于 `web/`）。

## 环境准备

- 按照 `README.md` 或 `README_zh.md` 完成依赖安装。
- 在项目根目录准备 `.env` 文件，并填入必需的 API Key 与配置。

## 启动方式

在项目根目录执行以下任一命令：

```bash
python cli.py web --host 0.0.0.0 --port 8001
```

或使用 uv：

```bash
uv run cli.py web --host 0.0.0.0 --port 8001
```

说明：

- `--host` 默认 `0.0.0.0`，`--port` 默认 `8001`。
- 仅本机访问可将 host 改为 `127.0.0.1`。

## 访问方式

- 本机访问：`http://127.0.0.1:8001`
- 远程服务器：使用服务器 IP 或域名 + 端口访问，例如 `http://<server-ip>:8001`

## 基本使用流程

1. 在 Run 面板填写 `User ID` 与 `Message`。
2. 点击 `Run` 发起任务，输出会实时显示在 Stream Output 区域。
3. 通过 `Pause/Resume` 控制输出渲染，`Auto-scroll` 控制是否自动滚动，`Copy All` 复制输出。
4. 在 Agents/Tools/Workflows 面板点击 `Refresh` 拉取最新列表。

## 常见问题

- 页面无法打开或空白：
  - 确认服务已启动且端口未被占用。
  - 检查终端是否报错，并确认浏览器访问地址正确。

- 运行任务失败或报错：
  - 检查 `.env` 中的 API Key 配置是否完整。
  - 查看服务端日志定位异常原因。

- 端口冲突：
  - 使用 `--port` 指定新的端口，例如 `--port 8080`。

- 远程无法访问：
  - 确认云服务器防火墙与安全组已放行端口。
  - 确认服务以 `0.0.0.0` 监听。

## 停止服务

在启动服务的终端中按 `Ctrl + C` 结束。
